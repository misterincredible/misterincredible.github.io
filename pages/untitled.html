General structure

If you write a HTML you need to use a basic structure. Open and close every tag. The structure of the opening tag should look like this: First the opening <, then the closing >. Between those you have to write the element you want to use, if you write a closing tag, do the same, but add a / in between the element and the <. Things which stand between the opening and closing tag will be interpreted as parts of it, so if you might open a div and forget to close it, the Browser will try to interpret everything under the opening div-tag as a part of the div.

CSS Syntax

CSS syntax is very easy, because you don't need a closing tag. There's not much to talk about, which is why we will just start with the Syntax. To understand everything clear, read carefully now: The main parts of a CSS rule are the properties, their values and the selector. The selector "selects" a part of an HTML-document, which needs to be eddited. The selector defines what needs to be changed and last but not least the value shows how much it needs to be changed. So the CSS-Syntax works like that: selector { property: value; }. By the way, there's two important kinds of selectors: the class selector and the element selector. You can find more about that in Unit 2 and our lexicon.
