How to style text and use CSS

What is CSS?

CSS is our stylesheet, everything we want our webpage to look like begins and ends with CSS. Normally we link a CSS-data to our HTML-document, but you are also allowed to write your CSS in the HTML. We will just cover CSS-data, which is linked to HTML. If you're asking, what to do with CSS, we will explain you. CSS is used to create geonemtric figures and styling text. Styling text

Styling text might be the easiest thing about CSS. To style your text, you need a property, which is comparable to an HTML-element. Each property needs to be connected to selector. The properties value stands just behind it (devided by an :) and closed up by a ;. After that the properties values get executed on the HTML-element of the selectors. To style text there are lots of propertys available, here the most important ones:

    color: changes the colour of your text

    background-color: changes the colour of your texts background (more about that down this site)

    width, height: changes width/height of your background

    font-family: changes the font-style

    font-size: changes the size of the font

    border: changes the kind of border your background uses

More usefull properties and how to use them can you find in our lexicon.

Creating rectangles

If you want to create a rectangle, you have to define a height and a width. If you don't see anything, don't wonder, because to make it visible, you have to add borders (more about this in Unit 2). After doing that you can see the rectangle and of course edit the background colour.
