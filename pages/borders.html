Borders

How you should use borders

Borders are a good way to style your HTML and one of the biggest parts of layouting your page and using CSS. Earlier we talked about borders and now you might ask what we were talking about, well here is the explanation to your question. Borders are just that, what they are called. They mark the borders of elements (more about those down this site) You can use them for multiple purposes. The Syntax is quite easy, because we are talking about CSS-properties. Remember the CSS-Syntax from Unit 1? Here we go, this is how to structure it

Some knowledge about elements

So, this is going to be very short. Talking about elements, there are 3 types: block, inline and inline-block. You might associate them with the layouting modes in MS Word. The inline-element is structured like the textbox in Word. This element is found, as the name says, inline. The block-element is comparable to the geonemetric figures in MS Office, it's found outline. A inline-block-element is a mixture of both things. It's a block element, which you can find inline.

Creating circles

This one is a little bit more complex. To understand it you got to try it yourself and think of geometry, think it was 9th class at high school. Remember how many corners a circle has? 0? No, that's wrong, a circle has infinitely corners. The difference might be small, but very important for the following lesson. In CSS and HTML a circle has 4 corners. Guess what comes next. In CSS a circle looks like rectangle. But which: it looks like a square. The circle is, like we just learned, a manipulated square. But how do you manipulate it? This is the moment when you need borders. Remember, a border shows where an element ends, what would happen if the border would be "wrong". Yeah! This is what we are going to do now. If we manipulate the radius of the border, the border-radius (new property), we will create a circle, set the border radius to 50% and see what happens. A circle! If you want to make other objects with round parts in it you can use the border-radius too. You have to set the radius for each corner in your rectangle itself. This property is called border-top/bottom-left/right-radius and is used just like the border-radius.
