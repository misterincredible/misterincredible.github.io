Listing with ul/li

How to create a list

Creating a list can be very clever. When you make menus or lists themselves you should use the ul-/li-items. With a ul you open and close a list. Each item of a list get's the li-item which opens and closes it. The list will automaticly create dots to open each li up. If you want to change that, you have to go to your CSS and tipe in our next new property list-decoration and watch the multiple decoration options like roman (roman numbers), ordinal (ordinal numbers) or of course none (nothing). Of course there are way more. Just look them up in the internet.
