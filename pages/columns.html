Columns

What are columns?

Columns are the best way to structure your website or design tables. Each site is devided in 12 columns, which run verticaly. Because you need special Add-Ons to let that work (they show the browser where the columns are), we can use another strategy. Now listen!!! Wrap the things you want to put, maybe next to each other; in a div and give them a class. Don't forget, the name should fit to that what it does. Now open up a new CSS-rule in your stylesheet. Add a float-property. Float your content to the left. If you finished add a max-width-property, and look how many objects you have. Look how many percent an object takes and then write this percentage behind max-width. Always keep in mind, that those values are relative and always will be used on the object, div, ect. the element wrap, which you want to edit. Don't forget to make it an inline-block-element.
